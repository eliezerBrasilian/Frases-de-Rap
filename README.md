
# Frases de Rap

Um aplicativo criado para fãns do gênero, e que contém diversas frases e trechos de música que foram cantados pelos mais famosos cantores dos seguintes gêneros: Rap, Hip-Hop e Trap.


## Tecnologias utlizadas

 - Jetpack Compose
